# Carlos Miyén Brandolino — Portfolio

Portfolio profesional bilingüe (ES/EN): gestión de operaciones + desarrollo de software.

**🌐 Sitio en vivo:** [miyobran.github.io](https://miyobran.github.io/) · [English version](https://miyobran.github.io/en/)

## Stack

Sitio estático hecho a mano, sin frameworks ni build:

- **HTML5** semántico
- **CSS3** — Custom Properties, Grid, Flexbox, `prefers-reduced-motion`
- **JavaScript** vanilla — Intersection Observer para animaciones de scroll
- **GitHub Pages** como hosting (rama `main`, raíz)

## Estructura

```
├── index.html          # Portfolio en español
├── en/index.html       # Versión en inglés
├── blog/               # Artículos y casos de estudio (ES)
├── css/styles.css      # Sistema de diseño (tokens en :root)
├── assets/             # Imágenes (badges de certificaciones)
├── js/main.js          # Animaciones de scroll
├── .nojekyll           # Desactiva el procesamiento Jekyll de Pages
└── AGENTS.md           # Guía de mantenimiento para agentes de IA
```

## Desarrollo local

No requiere instalación: clonar y abrir `index.html` en el navegador.

```bash
git clone https://github.com/MiyoBran/MiyoBran.github.io.git
```

## Mantenimiento

Las convenciones de contenido, el sistema de diseño y el procedimiento para agregar
proyectos o posts están documentados en [`AGENTS.md`](AGENTS.md).

La versión anterior del sitio (Jekyll) se conserva en la rama `jekyll-legacy`.
